# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2026-06-15

### Added
- Initial release of cortex-plugin-whatsapp-telegram
- `whatsapp_send` — Send WhatsApp messages
- `telegram_send` — Send Telegram messages
- `telegram_read` — Read recent Telegram messages
- `bridge_send_alert` — Send alerts to all configured platforms
- `bridge_status` — Check bridge configuration status
- `bridge_set_webhook` — Configure Telegram webhook
- WhatsApp and Telegram configuration UI
